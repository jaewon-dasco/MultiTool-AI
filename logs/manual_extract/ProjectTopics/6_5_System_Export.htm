System Export

 

This function exports all project files that include the device configuration to the project folder. MultiTool Creator automatically creates a folder structure where each device has its own folder. The folder is named after the device and contains all the all the device configuration data. The project is automatically saved when System Export is selected. 

 

To export the system, select  icon in the Network Editor or go to Project > System Export. The files are exported to the project folder and new folders are created as described above.

 

 

The exported files must also be imported to CODESYS.

When exported, a prefix is added to variable names. For more information about the prefixes, refer to sections Variable Prefixes and Fixed Variable Names

 

 

Read only files, such as libraries from version control, are not overwritten in System Export.
