Basic Skills Required 

The user of this document must have professional skills in machine controlling, CAN communication, PLCopen programming according to IEC61131 and should have skills to use CODESYS 2.3/3.5 programming environment.

Please refer to CODESYS 2.3/3.5 manual for further information on programming environment and required installations.

Please refer to CAN and CANopen documentation from CAN in Automation for further information on communication issues.
