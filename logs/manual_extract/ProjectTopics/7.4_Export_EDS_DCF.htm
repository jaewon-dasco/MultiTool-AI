Exporting EDS / DCF / XDD / XDC / XML

 

The device's Object Dictionary and PDO configurations can be exported to EDS (Electronic Data Sheet), DCF (Device Configuration File), XDD (XML device description), or XDC (XML device configuration) files. These file types are defined by the CANopen standards (CiA DS306 and CiA DS311). 

 

The EDS and/or XDD file describes the communication behavior and object dictionary of a non-configured CANopen device.

 

The DCF and/or XDC file also includes configured settings for a specific network, such as Object Dictionary and PDO settings.

 

To export one of these files:

  1. Open the device configuration and go to the Object Dictionary tab

  2. Select the CAN channel

  3. Select and the Export dialog opens

  4. Select the Export Type

  5. If you are exporting a DCF file, fill in the Last EDS field (use the device's EDS file name if there is such)

  6. Fill in the needed fields and click OK

  7. Save As dialog opens. Browse to the location, give the file name and select Save.
