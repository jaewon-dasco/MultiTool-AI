Fixed Variable Names

Fixed Variable Names are not applicable for S Series / E Series projects.

 

 

Some of the variables MultiTool Creator creates for CODESYS projects have fixed names. These variables provide information about hardware and are not user configurable.

 

The following table lists fixed reserved variable names:
