Setting Background for Network Editor

Set Background

To set a background image for MultiTool Creator Network Editor:

 

  1. Select Project > Editor > Background > Set Background

  2. Browse the image and select Open

 

MultiTool Creator copies the background image to the project.

 

Background Alignment

The background image can be aligned as follows:

 

  •  

  • Middle Top

  • Right Top

  • Left Center

  • Middle Center

  • Right Center

  • Left Bottom

  • Middle Bottom

  • Right Bottom

 

Clear Background

Select Project > Editor > Background > Clear Background to set default background.
