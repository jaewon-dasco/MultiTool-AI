Editing Network Events

 

To view all the events in the network, select Events from the Network Editor hover menu. This view shows the configuration for the event consumer and also a list of all network events.

 

To add a new event, select icon to pick the source device of the event. The event will be added to this device and it will also be seen on the device's Event tab.

To delete an existing event, select the event and then select icon.

To view and/or edit event levels, select icon. This will open the Project Preferences.
