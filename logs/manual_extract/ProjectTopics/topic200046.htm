Functional Versions

This section describes the different product variants, also known as functional versions. The functional version can be found on your control unit's product label. Unit's HW revision can be checked, for example, using Epec CANmoon software. For more detailed information about the product, refer to it's technical manual (hyperlinks to the manuals are provided in the product tables below).
