Safety Guidelines 

The software is developed and tested by the manufacturer to be used as a tool for controlling and configuring the manufacturer’s hardware. The software is designed to be used only for machine controlling purposes and is intended for professional use by properly trained personnel only.

 

The manufacturer does not assume any responsibility for the software being fit for any particular application, unless otherwise expressly stated in writing by the manufacturer.

 

The user of the software should always follow general machine safety guidelines, directives and regulation appropriate to his/her country or market area.

 

A separate safety analysis is always recommended for the machine and its control system. The features of this product should be well documented in machine and control system documents so that the machine operator has the right information on how to operate the machine correctly and safely.

 

The software does not compile any other certification, standard or test if not mentioned in this documentation.

 

The software is not to be changed, altered, modified or reverse engineered in any situation by anyone, but the manufacturer, unless otherwise agreed upon in writing.

 

The system used with this software should be designed and constructed according to the manufacturer’s general mounting and cabling instruction document.

 

The manufacturer reserves the right to improve, upgrade or alter its products without notice.
