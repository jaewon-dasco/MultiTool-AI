Editing Parameter CSV

Use CSV Editor view to edit some of the parameter’s properties, organize parameters into logical groupings and export edited structure into CSV format.

 

Before editing parameter CSV:

 

  1. Add the needed devices and configure the needed ODs for them (double-click the device and select the Object Dictionary tab)

  2. Add a network between the devices (Adding, Editing and Deleting Network)

To open the CSV Editor and edit the parameter CSV configuration, select a network and click icon on the hover menu.

 

The CSV Editor shows all parameters with Variable Type Parameter, Retain or Retain Persistent. When the editor is opened, parameters are updated automatically based on the network’s devices' OD definitions. Parameter / Group Name column has its own identifying colors. New parameters are designated with a blue color.

By default, only some of the parameter properties are shown. To show all the parameter columns, select icon.

Adding and Removing Groups

To add new groups, right-click on table and select Insert Group / Add Group Above / Add Group Below. The maximum amount of grouping levels is 4. The groups can be organized by drag-and-drop.

 

To remove a group, right-click on the group row and select Remove.

 

 

Editing and Organizing Parameters

To edit column values, double-click the cell and edit it.

 

To organize parameters, select the parameter/group and drag-and-drop it to a new group

 

 

Adding Extra Columns

To add extra columns to the parameter table, select icon and

 

  1. Extra Columns dialog opens

  2. Add New column

  3. Double-click the created column to give a name for it

  4. Select the Data Type

    • Note that the extra columns are not validated based on the Data Type

  5. Click OK and the new columns are added to the end of the table

 

 

Editing Access Levels

To edit parameter access levels, select icon and

 

  1. Edit Access Levels dialog opens

  2. It is possible to change the level value, name and number of levels in the dialog.

  3. Click OK to save changes and exit the dialog.

 

The given access levels are used in parameter table columns Read Access and Write Access, select icon to see the hidden columns and select the cell to change the access level.

 

 

Exporting CSV

To export the parameter CSV, select icon. Select the file location and click OK.

 

The parameter CSV file is automatically exported on System Export. The parameter CSV file name is automatically generated on System Export to the project directory with the network name.

 

 

 

Resetting to Defaults

To reset the parameter CSV to defaults, select icon. Reset reads the values from OD and updates the CSV table accordingly. Note that the values for Low, High and Default are synchronized also from CSV to OD, so reset will not affect them.
