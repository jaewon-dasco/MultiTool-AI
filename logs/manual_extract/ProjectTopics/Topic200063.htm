Recent Projects

Recent Projects is a list of previously used projects.

 

Recent projects list is specific for MultiTool Creator version. Recent projects list is not copied to a new MultiTool Creator version.

 

 

To open a recent project:

 

  1. Select File > Recent Projects

  2. Select the project on the list

 

To clear the recent project list, select File > Recent Projects > Clear.

 

To change the amount of recent projects in the list, go to File > Settings > Environment > Max Count of Recent Projects.
