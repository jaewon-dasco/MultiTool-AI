Preferences

 

This topic describes project preferences. To open the Project Preferences dialog:

 

  1. Select  Project > Preferences..

  2. To change the variable value, select the current value and type a new value.

 

Project preferences are described in the following table:

 

Preference

Description

Project Name

The project name that is given when the project is created.

It is not recommended to change this once the Create CODESYS Project has been done for some of the project devices. Renaming will create a new directory to which any user specific application code is not transferred.

 

Project Location

Path to the project directory

 

Default Array Start Index

The start index for new arrays that are created on the Object Dictionary tab. For example,

  • value 0 means that the arrays are defined to start from index 0 > [0..10]

  • value 5 means that the arrays are defined to start from index 5 > [5..10]

 

Maximum Index Size for Checksum

The maximum size of one OD index's data in bytes, needed for calculating the checksum.

Project Key

Project key is used for encryption in safety parameter authentication. See Epec Programming and Libraries Manual for more information. The same key is used for all control units in a MultiTool Creator project. By default, MultiTool Creator creates a random key.

Event Levels

Lists the event levels. MultiTool Creator defines three event levels as a default: Alarm, Warning and Notification. The default setting for logging and pop-up can be done here.

 

To add a new event level, select icon.

To remove an event level, select icon.

To reset the default event levels and their settings for logging and pop-up, select icon.
